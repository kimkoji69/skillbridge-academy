About SkillBridge Academy

Project Title: SkillBridge Academy – Learn & Earn Platform

Developed By: KYEWALYANGA ANILOV BAKER

Project Type: UCE ICT Project

Purpose: SkillBridge Academy is a web-based skill sharing platform designed to help individuals learn income-generating skills through educational video tutorials.

Target Users: Students, professionals, and entrepreneurs interested in learning practical money-making skills.

System Features:

  • Skill video tutorials
  • Search functionality
  • Professional upload form
  • User login page

Limitations: This platform is a prototype and does not include a live database or backend integration.

Future Improvements: Integration with a database, real user accounts, secure login system, and direct video uploads.
